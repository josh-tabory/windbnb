# windbnb is a responsive vue js app 
Uses: 
- vue router
- api service
- json-server

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Install server in new terminal to access database
```
npm install json-server
```

### Run json-server 
```
json-server --watch db.json
```
